>***使用jekyll编译github的gh-pages生成静态页面***

---
### 安装jekyll

```
brew install ruby

gem install jekyll

gem install bundler

gem install jekyll-paginate

gem install jekyll-gist
gem install bundler
bundler install
```
---
### 启动本地服务
```
bundle exec jekyll serve
```
---
### 生成静态页面
```
 bundle exec jekyll build
```
